# Team Xanthronz – Official Website ⚡🚗

This repository contains the official website of **Team Xanthronz**,  
the eBaja electric vehicle team of **NIT Agartala**.

The website showcases the team, projects, events, achievements, and gallery,  
built using modern frontend technologies with a strong focus on performance and responsiveness.

---

## 🚀 Tech Stack

- **React** (Vite)
- **Tailwind CSS**
- **React Router**
- **Framer Motion**
- **JavaScript (ES6+)**

---

## ✨ Features

- Fully responsive design (Mobile, Tablet, Desktop)
- Smooth animations and transitions
- Dynamic team (Squads) section by year
- Projects and innovations showcase
- Events listing
- Image gallery with slider and collage
- Clean layout with reusable components
